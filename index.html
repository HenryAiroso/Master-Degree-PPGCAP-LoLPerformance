About the project

Academic, non-commercial research project that analyzes MOBA matches similar to League of Legends using machine learning and deep learning. Match data is collected through the Riot Games API and processed offline to study team and player performance over time.

Data is used only in aggregate/anonymized form for the dissertation and possible scientific publications. No overlays, boosting, betting, or tools that give unfair competitive advantage.

  • Machine Learning
  • Game Analytics
  • Riot Games API
