Master’s Dissertation – Applied Computing

Master-Degree-PPGCAP-LoLPerformance

This site presents League of Legends Performance AI – Master’s Dissertation, a master’s degree project in Applied Computing. The research analyzes League of Legends matches using machine learning and deep learning to model team and player performance over time.

Project Goals

  • Collect historical ranked (and, when allowed, competitive) match data using the Riot Games API.
  • Build structured datasets with timelines (gold, XP, objectives, kills, etc.).
  • Train and evaluate predictive models (e.g., win probability over time, game state classification).
  • Produce aggregated, anonymized statistics for academic analysis and publications.

Tech Stack

  • Python – data collection, preprocessing, feature engineering.
  • Jupyter Notebooks – experiments, models, visualizations.
  • Riot Games API – account-v1, summoner-v4, match-v5.
  • Data Dragon / static LoL data – champions, items, runes.

Ethics and Data Use

  • Non-commercial, academic-only use.
  • Data is analyzed in aggregate; individual players are not identified.
  • No overlays, boosting, betting, or tools that provide unfair competitive advantage.
