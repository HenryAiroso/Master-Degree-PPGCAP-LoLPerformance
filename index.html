Project Overview

Master-Degree-PPGCAP-LoLPerformance is a non-commercial master’s dissertation project focused on League of Legends Performance AI. The research analyzes historical League of Legends matches using machine learning and deep learning to model team and player performance over time.

Goals

  • Collect historical ranked (and, when allowed, competitive) match data.
  • Build structured timelines (gold, XP, objectives, kills, etc.).
  • Train and evaluate predictive models (e.g., win probability, game state classification).
  • Produce aggregated, anonymized statistics for academic analysis and publications.

Riot Games API Usage

The project uses the official Riot Games API only for offline, research-oriented data collection:

  • account-v1 / summoner-v4 – resolve Riot IDs and PUUIDs to seed match collection.
  • match-v5 – retrieve match details and timelines for feature engineering and model training.
  • Data Dragon / static LoL data – champion, item and rune metadata.

Tech Stack

  • Python – data collection, preprocessing, feature engineering.
  • Jupyter Notebooks – experiments, models, visualizations.
  • Riot Games API – account-v1, summoner-v4, match-v5.
  • Data Dragon – champions, items, runes.

Ethics and Data Use

  • Non-commercial, academic-only use in a controlled research environment.
  • Data is analyzed in aggregate; individual players are not identified or profiled.
  • No overlays, scripts, bots, boosting, betting or tools that provide unfair competitive advantage.
