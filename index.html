Dissertação de mestrado em Computação Aplicada usando dados de partidas de League of Legends para pesquisa em machine learning.

Visão Geral do Projeto

Master-Degree-PPGCAP-LoLPerformance é um projeto de dissertação de mestrado não comercial focado em IA de Desempenho no League of Legends. A pesquisa analisa partidas históricas usando machine learning e deep learning para modelar o desempenho de equipes e jogadores ao longo do tempo.

Objetivos

  • Coletar dados históricos de partidas ranqueadas (e, quando permitido, competitivas).
  • Construir cronogramas estruturados (ouro, XP, objetivos, abates, etc.).
  • Treinar e avaliar modelos preditivos (ex: probabilidade de vitória, classificação de estado do jogo).
  • Produzir estatísticas agregadas e anonimizadas para análise e publicações acadêmicas.

Uso da API da Riot Games

O projeto usa a API oficial da Riot Games apenas para coleta de dados off-line e orientada à pesquisa:

  • account-v1 / summoner-v4 – resolve IDs e PUUIDs para iniciar a coleta de partidas.
  • match-v5 – recupera detalhes da partida e cronogramas para engenharia de recursos e treinamento de modelos.
  • Data Dragon / dados estáticos do LoL – metadados de campeões, itens e runas.

Tech Stack

  • Python – coleta de dados, pré-processamento, engenharia de recursos.
  • Jupyter Notebooks – experimentos, modelos, visualizações.
  • Riot Games API – account-v1, summoner-v4, match-v5.
  • Data Dragon – campeões, itens, runas.

Ética e Uso de Dados

  • Uso não comercial, apenas acadêmico em um ambiente de pesquisa controlado.
  • Os dados são analisados de forma agregada; jogadores individuais não são identificados ou perfilados.
  • Sem overlays, scripts, bots, boosting, apostas ou ferramentas que ofereçam vantagem competitiva injusta.
